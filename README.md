# Myapp

Package: `com.moby.app`  
Min SDK: 23

## Cara Pakai

1. Buka folder ini di **Android Studio**
2. Tunggu Gradle sync selesai
3. Edit `app/src/main/java/com/moby/app/MainActivity.java`
4. Edit layout di `app/src/main/res/layout/activity_main.xml`
5. Run di emulator atau HP

## GitHub Actions

Push ke branch `main` → otomatis build APK.
Download APK di tab **Actions → Artifacts**.
