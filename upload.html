🛠️ AndroidTools JSON Generator

Buat & kelola tools.json untuk repo Anda — by mondev
⚙️ Preset URL Global
💡 Tips URL: Jika Anda isi Base URL, kolom URL di setiap item cukup diisi nama file-nya saja, misalnya MyApp.apk — akan otomatis digabung saat Generate. Jika URL sudah diawali http, preset diabaikan.
