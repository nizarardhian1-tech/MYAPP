📤 Import JSON yang Sudah Ada Paste JSON lama → otomatis jadi form
💡 Cara kerja: Paste JSON lama → klik Import → semua otomatis jadi form yang bisa diedit. Tambah item baru, edit, hapus — lalu klik Generate JSON untuk hasil akhirnya. Tidak perlu pusing atur kurung kurawal atau koma lagi.
